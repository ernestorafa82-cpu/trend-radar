# FINAL SALE LISTING — Trend-Radar / Market Wow Radar

## Se vende
Asset operativo y transferible de **market intelligence diaria** que convierte ruido informativo en un **brief Top 10 accionable**, listo para operar, demostrar y transferir.

## Qué hace
Recopila señales públicas de mercado, reduce ruido, prioriza oportunidades y genera un radar diario en:
- Markdown legible para humano
- JSON estructurado para reutilización o integración

## Buyer ideal
- founders
- scouts
- micro-VCs
- equipos de innovación
- analistas de mercado
- product studios

## Qué problema resuelve
Reduce el tiempo y la fatiga de revisar múltiples fuentes para detectar señales útiles de mercado, research y startups.

## Qué incluye
- repositorio completo `trend-radar`
- lógica actual de generación del radar
- setup reproducible
- demo reproducible
- sanity check
- documentación de handoff y operación
- baseline publicado con tag `v0.1.0-transferable-asset`

## Qué no incluye
- cuenta OpenAI del vendedor
- billing/cuota de terceros
- infraestructura externa no pactada
- soporte indefinido
- desarrollo a medida posterior
- conversión a SaaS completa

## Estado actual
Activo funcional, demoable, documentado y transferible.

Está empaquetado como **asset operativo vendible**, no como plataforma SaaS madura.

## Modalidad de entrega
- repositorio GitHub como opción principal
- export `.zip` del repo si el comprador lo prefiere
- handoff corto opcional
- soporte limitado solo si se pacta aparte

## Rango orientativo
**2.500 € – 4.000 €** como base para modalidad **asset + handoff corto**.

## Por qué tiene valor
- ahorra tiempo de construcción al comprador
- ya está empaquetado y entendible
- ya tiene setup, demo y documentación
- ya tiene narrativa comercial clara
- puede operarse tal cual o servir como base para un producto mayor

## Nota de posicionamiento
No se vende como “startup con tracción”.
Se vende como **activo digital bien empaquetado, funcional y transferible**, con encaje claro para scouting, research y opportunity radar.
