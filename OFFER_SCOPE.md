# OFFER SCOPE — Trend-Radar / Market Wow Radar

## Qué se vende
Se vende un asset operativo y transferible de market intelligence diaria en formato script-first, con repo, documentación, setup reproducible, demo reproducible y baseline funcional ya validado.

## Qué incluye
- repositorio `trend-radar`
- lógica actual de generación del radar
- outputs en Markdown y JSON
- documentación actual (`README.md`, `HANDOFF.md`, `OPERATIONS.md`, `SALE_ASSETS.md`, `LISTING_DRAFT.md`, `DELIVERY_CHECKLIST.md`)
- scripts de setup, demo, sanity check y ejecución diaria
- tag publicado `v0.1.0-transferable-asset`

## Qué no incluye
- cuenta OpenAI del vendedor
- billing o cuota de terceros
- infraestructura externa no pactada
- desarrollo a medida posterior a la venta
- soporte indefinido
- garantías de rendimiento comercial o ingresos
- conversión a SaaS completa

## Modalidad de entrega propuesta
- transferencia del repositorio GitHub o export completo del repo
- walkthrough corto de handoff
- validación conjunta de setup/demo si se pacta
- cierre explícito del perímetro incluido

## Soporte post-venta propuesto
- opción A: sin soporte incluido
- opción B: soporte corto de transición durante un periodo limitado, si se vende aparte
- cualquier trabajo adicional queda fuera del precio base salvo pacto expreso

## Posicionamiento correcto
Este activo debe venderse como:
- asset operativo vendible
- base transferible para scouting / research / opportunity radar
- sistema ligero ya empaquetado y entendible

No debe venderse como:
- plataforma SaaS enterprise
- producto multiusuario maduro
- sistema autónomo sin dependencia operativa externa
